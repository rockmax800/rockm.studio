# Service Offerings

> Business Layer — What AI Production Studio delivers.

## 1 — Purpose

Defines the service categories available through the studio.

---

## 2 — Service Categories

| Category | Description | Typical Effort Band |
|----------|-------------|-------------------|
| Frontend Development | Next.js/React/Tailwind applications | small–large |
| Backend Implementation | API design, database schema, edge functions | medium–large |
| Product Scoping | Blueprint and estimation from brief | small |
| Code Review | Automated + founder review of existing code | small–medium |
| Bug Fix | Targeted issue resolution | small |
| Feature Addition | New capability on existing project | medium–large |

---

## 3 — Constraints

- All services follow the standard intake → blueprint → estimate → launch pipeline.
- No service bypasses the review and approval lifecycle.
- Pricing is based on estimated token usage and effort band.
